# IPL Cricket Analytics Dashboard

This project analyzes IPL cricket data using Power BI to generate insights about player performance, team statistics, and season trends.

## Tools Used
- Power BI
- Excel / CSV datasets
- Data Visualization

## Features
- Total runs, wickets, and matches KPI
- Top batsmen and bowlers analysis
- Team performance visualization
- Seasonal trend analysis
- Interactive filters

## Dataset
The dataset contains IPL match and delivery data including runs, wickets, teams, and player information.
